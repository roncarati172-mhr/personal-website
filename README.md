# Adam Smith - Personal Art Portfolio

A minimalistic, beautifully animated personal website showcasing artwork.

## Features

- **Minimalistic Design**: Clean, elegant interface with a sophisticated color scheme
- **Smooth Animations**: Powered by GSAP (GreenSock Animation Platform) for fluid, professional animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Modern Typography**: Clean, readable fonts with careful spacing
- **Scroll Animations**: Elements fade in and animate as you scroll
- **Interactive Gallery**: Hover effects and smooth transitions on artwork items

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (Vanilla JS)
- GSAP (GreenSock Animation Platform) - Animation library
- Google Fonts (Inter)

## Setup

1. Simply open `index.html` in a web browser
2. No build process or dependencies required - all libraries are loaded via CDN

## Customization

### Adding Your Artwork

Replace the placeholder artwork divs in the gallery section with your actual images:

```html
<div class="artwork-image">
    <img src="path/to/your/artwork.jpg" alt="Artwork Title">
</div>
```

### Color Scheme

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-bg: #0a0a0a;
    --secondary-bg: #1a1a1a;
    --text-primary: #ffffff;
    --text-secondary: #b0b0b0;
    --accent: #d4af37;
}
```

### Contact Information

Update the contact section in `index.html` with your actual email and social media links.
